# Best Kasi Kota App

A modern, user-friendly food ordering application designed for township eateries ("Kasi Kota" refers to popular South African township food). This app streamlines the ordering process for customers and provides comprehensive management tools for administrators.

## Features

- **Customer Interface**: Browse menu, add to cart, place orders, track orders.
- **Admin Dashboard**: Login, manage menu, inventory, orders, analytics, notifications.
- **Order Management**: Track and manage customer orders.
- **Inventory Management**: Monitor stock levels and logs.
- **Analytics**: Insights into sales and performance.
- **Notifications**: Real-time updates for orders and stock.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **UI Components**: Radix UI, Tailwind CSS
- **Backend**: Supabase (database, authentication, real-time)
- **Charts**: Recharts
- **Icons**: Lucide React

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/juwiijones984/Best-Kasi-Kota-App.git
   cd Best-Kasi-Kota-App
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:5173` (or the port shown in terminal).

## Build for Production

```bash
npm run build
```

## Project Structure

- `src/components/`: Reusable UI components
- `src/supabase/`: Backend functions and utilities
- `src/utils/`: Helper functions
- `src/styles/`: Global styles
